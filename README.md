## Project Overview

The **Tour & Event Planner System** is a collaborative Software Engineering project that focuses on the analysis and design of a unified tourism platform. The proposed system aims to help users discover tourism events, explore attractions, book activities, and manage their tourism experience through a centralized platform.

This project was developed as part of the **CSCE 232 – Software Engineering** course at the **College of Computer Science and Engineering, University of Ha'il**, following the Software Engineering lifecycle from requirements gathering to system design.

---

## Project Objectives

- Analyze stakeholders and user needs.
- Gather and document software requirements.
- Define functional and non-functional requirements.
- Design a scalable tourism event management system.
- Model the proposed solution using UML diagrams.
- Produce professional Software Engineering documentation.
